AGMTestBox

Overview

Controls communication with the AGM DU Test Box.

1.1 SendAGMPacket

1.1.1 Header

int DLLEXPORT __stdcall SendAGMPacket(char *port, int baud, char *command, int command_size, char *status_message)

1.1.2 Purpose

Send a control packet to the AGM Test Box through an RS-232 COM port and reads the response from the device.

1.1.3 Parameters

Parameter Type Description
* port char COM port name.
baud int Communicating baud rate.
*command char * Command sent to the AGM Test Box.
command_size int Number of bytes in the command.
*status_message char * Stores the device response or error message.

1.1.4 Return Value

Error Description
AGM_SUCCESS command completed successfully.
AGM_ERR_INVALID_PARAMETER A parameter is invalid or missing.
AGM_ERR_OPEN_PORT The COM port could not be opened.
AGM_ERR_TIMEOUT No response was received before timeout.
AGM_ERR_READ_FAILURE The response could not be read from the COM port.

1.1.5 Example

1.2 RIBVideoControl

1.2.1 Header

int DLLEXPORT __stdcall RIBVideoControl(int enable, char *port, int baud, char *status_message)

1.2.2 Purpose

Enables or disables the RIB video output on the AGM Test Box by sending the appropriate control command the COM port.

1.2.3 Parameters

Parameter Type Description
enable int Selects the video output state: 1 enables and 0 disables.
port char * COM port name used to communicate with AGM Test Box.
baud int Communication baud rate.
status_message char * Stores the device response or error message.

1.2.4 Return Value

Error Description
AGM_SUCCESS Video control command completed successfully.
AGM_ERR_INVALID_PARAMETER The enable value is not 0 or 1, or another parameter is invalid.

1.2.5 Example

1.3 AGMPowerControl

1.3.1 Header

int DLLEXPORT __stdcall AGMPowerControl(char* state, int plug, char* IP_address, char *status_message)

1.3.2 Purpose

Controls the power state of the AGM Test Box through the WTI Power Plug using an HTTP command.

1.3.3 Parameters

Parameter Type Description
state char* Power state. Valid values are "ON", "OFF", or "BOOT".
plug int Plug number. Valid values are ‘3’ (Top AGM) or 4 (Bottom AGM).
IP_address char * IP address of the WTI Power Plug.
status_message char * Stores the operation status or error message.

1.3.4 Return Value

Error Description
0 Power command completed successfully.
1 Invalid power state.
2 Invalid plug number.
3 Invalid IP address.

1.3.5 Example
