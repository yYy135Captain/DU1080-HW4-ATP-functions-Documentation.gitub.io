int DLLEXPORT __stdcall PhotometerMeasure(unsigned int com_port, double *luminance, double *chromaticity_x, double *chromaticity_y, double *u_prime, double *v_prime)

1.40.2 Purpose

Measure luminance, chromaticity x&y, u/v prime and display on photometer.

1.40.3 Parameters

Parameter Type Purpose
com_part unsigned int port to measure from
luminance double * luminance value (Lv)
chromaticity_x double * chromaticity x coordinate
chromaticity_y double * chromaticity y coordinate
u_prime double * scaled factor of chromaticity_x
v_prime double * scaled factor of chromaticity_y

0 Success
