PickeringRelay

Overview

The PickeringRealy module provides an interface for controlling a Pickering relay card through the Pickering PXI driver library. It supports initializing the relay card, changing the state of individual relay channels, and closing the relay session after operations are complete. The module uses the Pickering PXI driver to communicate wih the relay hardware.

1.42 PickeringRelayInitialize

1.42.1 Header

int DLLEXPORT __stdcall PickeringRelayInitialize(char *visa_resource_name, ViSession *session_handle, ViBoolean *test_result, char *status_message);

1.42.2 Purpose

Initializes the Pickering relay card and creates a communication session using the specified VISA resource.

1.42.3 Parameter

Parameter Type Description
visa_resource_name char * visa resource name
session_handle ViSession Returns the created session handle.
test_result ViSession Either VI_TRUE or VI_FALSE to indicate initialization output.
status_message char * Stores the initialization status and error message.

1.42.4 Return Value

Error Description
0 Relay card initialized successfully.

1.43 PickeringRelayAction

1.43.1 Header

int DLLEXPORT __stdcall PickeringRelayAction(ViSession *session_handle, char *action, int channel_id, char *status_message);

1.43.2 Purpose

1.43.3 Parameter

Parameter Type Description
session_handle ViSession Handle to the initialized Pickering relay session.
action char * Requested relay action
channel_id int Relay channel number to control.
status_message char * Stores status or error message.

1.43.4 Return Value

Error Description
0 Relay operation completed successfully.
-1 Failed to change the relay state.

1.44 PickeringRelayClose

1.44.1 Header

int DLLEXPORT __stdcall PickeringRelayClose(ViSession *session_handle, char *status_message);

1.44.2 Purpose

Closes the Pickering relay session and releases the associated hardware resources.

1.44.3 Parameter

Parameter Type Description
session_handle ViSession Handle of the active Pickering relay session.
status_message char * Stores status or error message.

1.44.4 Return Value

Error Description
0 Relay session closed successfully.
