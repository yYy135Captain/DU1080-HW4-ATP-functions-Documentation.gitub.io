Relay

Overview

The Relay library provides functions for controlling NI switch hardware used in the DU1080 HW4 ATP system.

It supports initialization, relay switching, signal routing, and discuss signal control.

1.53 RelayInitialize

1.53.1 Header

int DLLEXPORT _stdcall RelayInitialize(char *resource_name, ViSession *session, ViBoolean *test_result, char *status_message);

1.53.2 Purpose

Creates a switch session and initializes the relay module.

1.53.3 Parameter

Parameter Type Purpose
resource_name char * Switch resource name.
session ViSession Returns the switch session.
test_result ViBoolean Indicates success or failure.
status_message char * Returns status messages.

1.53.4 Return Value

Error Description
0 Initialization completed successfully.
1 Failed to open VISA resource manager.
2 Failed to open VISA resource.

1.54 RelaySet

1.54.1 Header

int DLLEXPORT _stdcall RelaySet(ViSession session, char channel[], int operation);

1.54.2 Purpose

Changes the state of a specified relay channel.

1.54.3 Parameters

Parameter Type Purpose
session ViSession Active switch session.
channel char[] Relay channel number.
operation int Relay state (NC OR NO).

1.54.4 Return Value

Error Description
0 Relay state changed successfully.
<0 NI Switch API error.

1.55 RelayClose

1.55.1 Header

int DLLEXPORT _stdcall RelayClose(ViSession session, char *status_message);

1.55.2 Purpose

Resets the switch module and closes the active session.

1.55.3 Parameters

Parameter Type Purpose
session ViSession Active NI Switch session to be closed.
status_message char * status message represented by string.

1.55.4 Return Value

Error Description
0 Session closed successfully.
<0 Failed to reset or close the session.

1.56 ReplayOpenAll

1.56.1 Header

int DLLEXPORT _stdcall RelayOpenAll(ViSession session);

1.56.2 Purpose

Disconnects all relay paths on the active switch module.

1.56.3 Parameters

Parameter Type Description
session ViSession Active switch session.

1.56.4 Return Value

Error Description
0 All relays opened successfully.
<0 Failed to open all relays.

1.57 RelaySignalSet

1.57.1 Header

int DLLEXPORT _stdcall RelaySignalSet(ViSession session, char *signal_name, void **signal_list, char *status_message);

1.57.2 Purpose

Set the relay state for a specified signal in the signal list.

1.57.3 Parameters

Parameter Type Description
session ViSession Active switch session.
signal_name char * Name of the signal to change.
signal_list void ** List of available relay signals.
status_message char * Stores the operation status message.

1.57.4 Return Value

Error Description
0 Signal state updated successfully.
-1 Signal list has not been loaded.
-2 Invalid relay operation.
-3 Signal was not found.

Parameter Type Description

test_result ViBoolean * Return test status.
status_message char * Stores status message.

1.58.4 Return Value

Error Description
